This plugin allows users to generate AI-based content and post it directly to their Facebook Business Page.

== Description ==
AI Content to Facebook Business integrates OpenAI's powerful AI capabilities with Facebook, enabling WordPress users to create engaging content for their Facebook Business Page directly from their WordPress admin panel. Features include AI-generated content suggestions and manual or automated posting options.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/ai-social-media` directory, or install the plugin through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->AI to Facebook screen to configure the plugin.
4.  Go to https://aigrowthguys.com/ai-social-media-wordpress-plugin/ for full instructions and how to connect Facebook and ChatGPT api's.

== Frequently Asked Questions ==
= Can I customize the AI-generated content before posting? =
Yes, you can edit the AI-generated suggestions before posting to Facebook.
